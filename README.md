# GMC

**G Money Core** is a personal research and engineering project in systematic trading.

This repository is a placeholder. Its previous contents — a specification for a planned trade-signal service, including an access-tier table and a pricing line — have been withdrawn pending specialist legal review.

There is no service, no channel, no subscription, and nothing is offered for sale.
